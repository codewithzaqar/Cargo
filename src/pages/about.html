A browser with almost no UI.

Cargo is a browser for people that live on the internet and hate mice. Cargo can be controlled using only a few keyboard shortcuts.

I built cargo since I spend a lot of time in the browser and did not find myself using most of the features. Cargo is built using electron - a famework for building cross platform apps using html, css and js - so cargo is a browser built using web technology.

Thanks for using cargo!

Keyboard shortcuts

Cargo is made for being used with the keyboard.

alt Open menu
command + shift + d Open devtools
command + left Go black
command + right Go forward
command + r Reload
command + h Go home
